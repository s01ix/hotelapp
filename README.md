# System Rezerwacji Hotelowych


## Opis
System umożliwiający zarządzanie rezerwacjami pokoi hotelowych. 
System będzie umożliwiał użytkownikom przeglądanie dostępnych pokoi, sprawdzanie
ich szczegółowych informacji oraz dokonywanie rezerwacji na wybrany termin. 
Użytkownicy będą mogli zakładać konta, logować się do systemu oraz zarządzać swoimi rezerwacjami. 
System będzie również umożliwiał realizację płatności online za pomocą zewnętrznego systemu płatności 
oraz obsługę płatności offline zatwierdzanych przez administratora.
